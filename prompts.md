# Prompts

## Definition

For each of these Esperanto words, print the word then a brief English translation for a glossary.

## Form extraction and definition

This is a list of possible Esperanto words in various forms, including noun forms like plural or object and various verb forms like participles. For each valid Esperanto word, write the most basic dictionary form (singular subject noun or adjective, verb infinitive, or adverb) along with a brief definition for a glossary that I am making. The glossary can include proper nouns as long as they are in valid Esperanto form.

## Translation check

The English text was translated into the Esperanto text. Proofread the translation and look for errors.
